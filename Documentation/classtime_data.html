Detailed Description

Time ADT.

This is an ADT that is able to store hours, minutes and seconds. It is also possible to do simple operations and change or get the stored values.

Set Time.

This function has two mandatory parameters. The amount of hours and minutes. Most of the time the amount of seconds is irrelevant; it defaults to 0.
