Detailed Description

Signaltrenght Bars.

This is a class which makes it possible to represent a signalstrength through bars. They try to look like signalbars known from mobile phones. They are completely customizable; the position, width, height amount of bars and spacing between bars.

auto signalWindow = hwlib::window_part(oled, hwlib::xy(108, 0), hwlib::xy(128, 15));
auto signalIndicator = signalBars(hwlib::xy(110, 8), 2, 2);
signalIndicator.print(window, radio.signalStrenght() / 12);
